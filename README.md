# TOP Sneakers

A small e-commerce style React app for browsing sneakers, adding items to cart, saving favorites, and creating orders. Built as a portfolio project.

## Features

- Browse sneakers catalog
- Search by name
- Add/remove items from cart
- Save/remove favorites
- Create an order (MockAPI)
- Skeleton loading states
- Responsive footer layout

## Tech stack

- React
- React Router
- Axios
- SCSS + CSS Modules
- MockAPI (as a demo backend)

## Getting started

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Deployment (Netlify)

- Build command: `npm run build`
- Publish directory: `build`
- SPA routing: included via `public/_redirects`

## Notes

This project uses MockAPI endpoints for demo purposes. If the API is unavailable, the UI will show an error state.
